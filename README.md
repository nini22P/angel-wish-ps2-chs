# angel-wish-ps2-chs

Chinese-Simplified localization toolchain for `Angel Wish 〜君の笑顔にチュッ！〜`

## Usage

1. Put the `Angel Wish - Kimi no Egao ni Chu! (Japan).iso` file in the `raw` folder.
2. Run `sh pack.sh` on terminal.
3. See `build` folder for the output.
